# ThumbsUp Family Rewards Support

Static support and privacy page for ThumbsUp Family Rewards.

The page includes app screenshots under `assets/screenshots/` for the onboarding,
kid dashboard, parent dashboard, and rewards views.

Expected GitHub Pages URL:

```text
https://jianhuichou.github.io/thumbsup-support/
```

## Publish

Create a public GitHub repository named `thumbsup-support` under the `jianhuichou` account, push this folder to the `main` branch, then enable GitHub Pages from the `main` branch root.
